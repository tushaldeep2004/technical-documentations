Introduction

It offers versatility, supporting multiple programming paradigms, including procedural, object-oriented, and functional styles. JavaScript's integration with the Document Object Model (DOM) allows developers to manipulate web page elements, enabling dynamic updates and user interactions. JavaScript

JavaScript (JS) is a versatile programming language widely used in web development. It runs directly in web browsers, allowing for client-side scripting. JavaScript's flexibility supports different programming styles, and it integrates with the Document Object Model (DOM) for dynamic web page manipulation.

With the emergence of technologies like Node.js, JavaScript can also be used for server-side development. Frameworks such as React, Angular, and Vue.js enhance JavaScript's capabilities for building complex user interfaces and managing application state. JavaScript has even found use in mobile app development through frameworks like React Native and Ionic.

As a high-level, interpreted language, JavaScript enables rapid development cycles and quick prototyping. It continues to evolve, driving innovation in web development and offering developers the means to create engaging and interactive experiences across platforms.


History

JavaScript (JS) was created by Brendan Eich at Netscape Communications Corporation in 1995. Originally called "Mocha" and later renamed "JavaScript," it was designed to enhance web page interactivity. In 1997, ECMAScript 1, the first standardized version, was released by Ecma International.

JavaScript underwent significant evolution over the years. ECMAScript 3 (1999) improved the language's robustness, while ECMAScript 5 (2009) introduced new features like strict mode and enhanced array manipulation. ECMAScript 6 (2015) was a major milestone, bringing modern syntax, classes, modules, and other powerful language features. Subsequent versions, including ES7 (2016) to ES11 (2020), added more enhancements and capabilities to JavaScript.

The popularity of JavaScript skyrocketed with the emergence of web applications and the rise of frameworks like jQuery, AngularJS, React, and Vue.js. It expanded beyond the browser with the introduction of Node.js, enabling server-side development using JavaScript. Today, JavaScript is widely used in web development, powering numerous websites and applications. Its continuous improvements, open-source nature, and broad adoption have established JavaScript as a versatile and influential programming language.

JS Expressions

  • Variables

    • var - Declares a variable with function scope.
    • let - Declares a block-scoped variable.
    • const - Declares a block-scoped constant variable.

  • Data Types

    • Number - Represents numeric values.
    • String - Represents textual data.
    • Boolean - Represents a logical value (true or false).
    • Array - Represents an ordered list of values.
    • Array - Represents an ordered list of values.

  • Operators

    • Arithmetic Operators: +, -, *, /, %.
    • Assignment Operators: =, +=, -=, *=, /=, %=.
    • Comparison Operators: ==, ===, !=, !==, >, <,>=, <=.
    • Logical Operators: && (AND), || (OR), ! (NOT).

  • Control Flow

    • "if" statement - Executes a block of code if a condition is true.
    • "else" statement - Executes a block of code if the preceding "if" condition is false.
    • "else if" statement - Adds additional conditions to be checked if the preceding conditions are false.
    • "switch" statement - Evaluates an expression and executes different blocks of code based on different cases.
    • "for" loop - Repeats a block of code for a specified number of times.
    • "while" loop - Repeats a block of code while a specified condition is true.

  • Functions

    • "function" keyword - Declares a function.
    • "return" statement - Specifies the value to be returned from a function.
    • Arrow Functions - A concise syntax for writing functions.

  • DOM Manipulation

    • document.getElementById() - Retrieves an element by its ID.
    • document.querySelector() - Retrieves the first element that matches a specific CSS selector.
    • element.innerHTML - Modifies the HTML content of an element.
    • element.addEventListener() - Attaches an event handler function to an element.

Examples

The exampes for the above exppressions:

Declaring Variables:

var name = "John";
let age = 25;
const PI = 3.14;

Using Operators:

let x = 10;
let y = 5;
let sum = x + y;
let product = x * y;
let isGreater = x > y;
console.log(sum); // Output: 15
console.log(product); // Output: 50
console.log(isGreater); // Output: true

Control Flow:

let hour = new Date().getHours();
if (hour < 12) {
console.log("Good morning!");
} else if (hour < 18) {
console.log("Good afternoon!");
} else {
console.log("Good evening!");
}

Functions:

function greet(name) {
console.log("Hello, " + name + "!");
}
greet("Alice"); // Output: Hello, Alice!
greet("Bob"); // Output: Hello, Bob!

DOM Manipulations:

let element = document.getElementById("myDiv");
element.innerHTML = "Hello, World!";
